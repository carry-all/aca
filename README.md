### [Meeting Timer](https://carry-all.github.io/aca/timer.html)
Timer for meetings

**Hotkeys:**
* `R` - Reset and start again
* `Space` - Start/pause
* `↑` / `↓` - Adjust by 10 secs
* `F` - Fullscreen
* `E` - Edit message
